# BeatCash — Product Context

## Product Purpose
Desktop tool for music producers and beatmakers to publish their beats simultaneously on YouTube and Instagram. Single-click upload: one source video/audio, two platforms, auto-adapted formats.

## Users
Solo music producers, beatmakers, and independent artists. They work late at night in a dark studio, monitor close, headphones on. They're not tech-savvy but they're aesthetic-sensitive — they care deeply about how tools look and feel. They want to feel like they're using a pro tool, not a school project.

## Register
product

## Brand / Tone
Bold but precise. Not loud. The Warhol dollar sign is the logo — it's camp but confident. The tool should feel like it belongs next to GarageBand and Logic Pro X on a Mac. Not a web app trapped in a window — a real desktop app.

## Anti-references
- Generic SaaS dashboards (Notion-style, cream + teal)
- Bootstrap-looking Python GUI tools
- Anything that looks like a tkinter default
- Neon cyberpunk (too cliché for music tools)
- Heavy glassmorphism decorative blur

## Strategic principles
- Speed over features: the publish flow must feel instant
- Confidence: the UI signals that the tool works before the user even clicks
- Dark-first: producers work at night in dim rooms
